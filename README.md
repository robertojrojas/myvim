# myvim
My vim configuration for MacOSX, Ubuntu, and Redhat/CentOS.

# To install

- git clone git@github.com:robertojrojas/myvim.git .vim

- cd $HOME

- mv ./vim/vimrc .vimrc

- modify .vimrc to fix paths for GO, etc....

- cd ./vim/bundle/YouComplete

- git pull

- ./install.py --clang-completer

- enjoy!
